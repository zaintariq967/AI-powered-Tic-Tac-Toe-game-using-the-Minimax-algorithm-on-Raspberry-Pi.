# AI-powered-Tic-Tac-Toe-game-using-the-Minimax-algorithm-on-Raspberry-Pi.
Developed a Tic-Tac-Toe AI using the Minimax algorithm on Raspberry Pi. The system evaluates all possible moves and selects the optimal action for the AI player, demonstrating core concepts of game theory and adversarial search. Includes a visual board and real-time best move prediction using Python.
